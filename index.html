What is Sky?

Sky is an open source database used for flexible, high performance analysis of behavioral data. For certain kinds of data such as clickstream data and log data, it can be several orders of magnitude faster than traditional approaches such as SQL databases or Hadoop.

The performance of Sky comes from optimized data organization, fast query execution, and the embarrassingly parallel nature of behavioral data. On a typical laptop you can expect to query at least 10 million events per core per second.

Demo

If you click the button below, you'll be able to interact with Sky using the UI called Skybox. Skybox allows you to explore the behavioral flow of the objects you're tracking in your database. In this example, we've loaded a small amount of the GitHub Archive's data set.

In this demo you can view what actions GitHub users perform on public, open source projects. For example, you can step through and see what users do after a respository “push”. Then you can drill down and see what users do after they do a “push” and then “create” a new repository. You can continue to drill down to any depth.

Getting Started

The best way to get up and running with Sky is to read through the Documentation and Getting Started pages. Sky only requires ZeroMQ as a dependency so it's easy to get up and running.
